# NitroCraft

NitroCraft is a modern rewrite of Crafatar built on top of Nitro Server.  
The project focuses on improving performance, maintainability, and scalability while providing a clean and reliable implementation of Crafatar's functionality.

This rewrite aims to modernize the original project with better architecture, improved efficiency, and a developer-friendly codebase.


## Goals

- Improve performance and scalability
- Provide a cleaner and more maintainable codebase
- Modernize the original Crafatar implementation
- Build a solid foundation for future features
